## GUIDES

### How to Add New Pages to the Website
1. Create a new HTML file in the project directory.  
2. Add a link to the new page in the site’s navigation or relevant page.  
3. Save, commit, and sync your changes.  
4. Refresh to make sure your changes have been saved.

### How to Add New Images to the Website
1. Put the image file in the appropriate folder (e.g. `/images`).  
2. In your HTML, reference the image using a relative path as defined below.  
3. Add any alt text describing the image for accessibility.  
4. Save, commit, and sync your changes.
5. Refresh to make sure your changes have been saved.


---

## DEFINITIONS

### Alt Text  
A text description of an image used in case an image does not load or the user needs accessibility allowances.

### GitHub  
A web-based place to store and manage your coding projects.

#### Comments  
Notes or explanations you share with your team to describe changes you’ve made.

#### Commit  
The second step in the GitHub commit process when you send over your files or other content to your repository.

#### Stage  
The first step in the GitHub commit when you prepare your changes to send or commit to your repository. 

#### Sync  
Final step in the GitHub commit when you make sure your changes have been added.

### Relative Path  
A link or file path that describes where a file is in your directory.

### Root Folder  
The main directory of the project that contains all subfolders and files. All folders build off of this root folder.
